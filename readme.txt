# AutoConcierge

Backend project built with Java and Spring Boot.

This project represents a basic structure for an automated concierge system that can process user requests and handle backend logic.

## Features
- Layered architecture (Controller / Service / Repository)
- DTO and Entity separation
- REST API structure
- Application configuration

## Tech Stack
- Java
- Spring Boot
- Maven

## How to run
1. Clone the repository
2. Run:
   mvn spring-boot:run
